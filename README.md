# Task Manager App

## Overview
The Task Manager App is a simple and intuitive application to help users manage their tasks efficiently. You can **add, update, delete, and track tasks** in a structured way, making productivity easier.

## Features
- Add new tasks with title and description
- Mark tasks as completed or pending
- Edit or delete existing tasks
- Responsive UI for better user experience
- Demo video included to showcase functionality

## Technologies Used
- **Frontend:** React.js, HTML, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (or SQLite/your database)
- **Version Control:** Git & GitHub
